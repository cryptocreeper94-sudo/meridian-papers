## Overview

Meridian is a four-layer deterministic architecture for wireless energy routing at building, campus, and planetary scale. The system treats energy delivery as a routable, identity-addressed resource — not a broadcast field — enabling deterministic paths from source to device with formal safety guarantees.

This repository contains the nine academic papers constituting the Meridian Canon (Canon 3 of the DarkWave Studios intellectual ecosystem) and their collected edition, *The Routed World*.

## Key Concepts

- **Deterministic Routing** — Energy beams follow computed, identity-addressed paths rather than broadcast radiation
- **Anchor Nodes** — Fixed structural transmitters forming a ceiling-plane mesh
- **Safety Envelope** — Real-time exposure density monitoring with automatic beam cutoff
- **Beam Geometry** — Five geometry types: Linear, Curved, Multinode, Distributed, Adaptive
- **4/42 Fractal Law** — Universal structural signature across all deterministic cognitive architectures
